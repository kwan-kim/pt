  • # 고객 관리시스템


    - 개요
    React.js 로 만든 고객관리시스템 웹 앱 프로젝트,
    고객 데이터 DB 와 서버는 Firebase - Google(Realtimebase) 사용
    서버와의 통신은 axios 를 이용, firebase 에 CRUD 를 api 로 만들어 사용 ( store.js )
    redux 를 사용해서 고객데이터 상태관리 처리 (redux , react-redux)
    React.js , css , google font , fontawesome 사용한 UI개발 (반응형,mobile)
    react-router-dom 을 활용한 SPA router 처리
    배포 : firebase hosting 사용한 빌드 배포

    - 구성
    ㆍ npx create-react-app customer-manager ( react v16.8.6 )
    ㆍ redux , react-redux
    ㆍ axios
    ㆍ firebase

  • # 실시간 유저 1:1 심리전게임


    - 개요
    1인개발 로 만든 JavaScript 웹 기반 하이브리드 앱, js 의 실력향상 과, 서버(Node.js) 공부 겸 시작해서, 최종 native 형태의 하이브리드 앱 으로 배포

    - 명칭 : 치트필드

    ㆍ JS , HTML5 , CSS
    ㆍ 서버는 node.js - Express 로 구현
    ㆍ 유저간 통신은 Socket.io 를 사용
